# RemoveControlDesktopPythonPyQt5
Đề tài: **Phần mềm điều khiển máy tính dành cho Window**

Phần mềm hỗ trợ điều khiển máy tính đơn giản trong mạng cục bộ (LAN) với 2 chế độ **Controller** và **User** trong đó:
    - Controller là máy tính điều khiển User từ xa.
    - User là máy tính bị điều khiển bởi Controller.

1. Hướng dẫn sử dụng:
**Cách 1**
Truy cập thư mục dist và chạy tệp tin run.exe 

**Cách 2** 
Mở cmd đường dẫn tới folder file VD: ...\RemoveControlDesktopPythonPyQt5
Nhập lệnh sau:
```bash
cd dist
start run.exe
```

2. Các lỗi thường gặp
