Hướng dẫn chạy file
- truy cập folder dist chạy file run.exe

Fix lỗi không chạy được file run.exe
- Cần tải python (https://www.python.org/) và thêm môi trường (add enviroment trong lúc install python)
- Mở CMD và chạy lệnh py -m pip install --upgrade pip

Nếu lỗi không thể tải được thư viện PyQt5 thì bạn phải lên trang chủ và xem hướng dẫn cách tải thủ công
